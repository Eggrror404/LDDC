## 功能

- [x] 搜索QQ音乐、酷狗音乐、网易云音乐的单曲、专辑、歌单
- [x] 一键下载整个 专辑、歌单 的歌词
- [x] 一键为本地歌曲文件匹配歌词
- [x] 支持保存为多种格式(逐字lrc、逐行lrc、增强型lrc、srt、ass)
- [x] 双击预览歌词,直接保存
- [x] 多种类型(原文、译文、罗马音)歌词随意合并
- [x] 保存路径多种占位符随意组合
- [x] 支持打开本地的加密歌词
- [x] 多平台支持
