# LDDC

> 精准歌词下载解密转换

## 功能

- [x] 根据歌曲名搜索歌词
- [x] 下载转换预览歌词
- [x] 合并歌词：支持精准歌词原文、译文与罗马音合并
- [x] 保存歌词：支持保存为lrc格式
